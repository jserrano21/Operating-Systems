One function of the operating system is Memory Management. It is in charge of the primary and main memory of the system meaning it tracks how many bytes of memory are being used by which user program, It Allocates the memory to a process when the process requests it and deallocates the memory when the process has terminated or is performing an I/O operation. 
Another function of the operating system is file management. A file system is used for better organizaton and easy navagation. It keeps track of status, uses, locations, and information. Lastly, it controls the resources.
